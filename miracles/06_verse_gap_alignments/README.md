## Methodology

### Counting Rules

1. **Text Standard**: Ḥafṣ ʿUthmānī (Tanzil format)
2. **Exclusive counting**: Excludes start verse, includes end verse (default)
3. **Inclusive counting**: Both endpoints where noted
4. **All calculations**: Verified against standard verse counts

### Statistical Approach

- **Universe**: 6,236 total verses in Quran
- **Probability formula**: P = (M-(d+1)) / C(M,2) where M = total verses, d = gap distance
- **Conservative estimates**: Account for selection potential
