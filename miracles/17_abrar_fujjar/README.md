## Methodology

### Counting Rules

1. **Specific terms only**: الأبرار (definite plural) and الفجار (definite plural)
2. **Count tokens** (not once-per-verse)
3. **Clitics allowed**: Prefixes like ك (ka-), ل (li-) included
4. **Exclude other forms**:
   - From root ب-ر-ر: برّ, البر, بارّ, بررة (piety, dutiful, righteous angels)
   - From root ف-ج-ر: فجر, فجارته (dawn, explosion)

- **الأبرار (Al-Abrar)**: The righteous, pious ones who fulfill obligations
- **الفجار (Al-Fujjar)**: The wicked, those who transgress divine boundaries
